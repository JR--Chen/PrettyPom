##快速格式化Maven项目pom文件工具

### 主要功能：
1. 自动寻找父类pom,格式化重复引入的parent依赖和import依赖
2. 将单独dependency中单独定于的依赖规整到management中
3. 去取management中的多余依赖（引发依赖冲突需手动解决）

### 待实现功能：
1. 更新插件中Java编译版本
2. 多import支持
2. 支持dependency中不同scope
3. 引入异常
